# Norynth Developer Quickstart

## 1. Run Local Node
```bash
ignite chain serve
# Access API: http://localhost:1317
# Swagger Docs: http://localhost:1317/docs
```

## 2. Key Endpoints
- **Update Params**: 
  `POST /norynth.norynth.Msg/UpdateParams` ([Docs](/docs))
- **Query Params**: 
  `GET /norynth/params`
